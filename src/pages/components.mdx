# Components

<nav style={{maxWidth: `var(--container)`}}>
Components are just HTML elements combined with global system classes. Here are some example recipes.

- `<App/>`
- `<Card/>`
- `<ContentAsideGrid/>`
- `<ContentGrid/>`
- `<Fieldset/>`
- `<Figure/>`
- `<Navlink/>`

---
</nav>

## `<App/>`

```jsx
<div>
  <header className="header">
    <h4 className="site-title">Site Title</h4>
  </header>
  <main className="main padding">
    <h4>Main</h4>
  </main>
  <footer className="footer">Footer</footer>
</div>
```

---

## `<Card/>`

```jsx
<div
  className="card"
  style={{
    // For demonstration
    maxWidth: `400px`,
  }}
>
  <h4 className="card-title">Card Title</h4>
  <Image />
  <p className="card-text">Card text.</p>
</div>
```

## `<ContentAsideGrid/>`

## `<Navlink/>`

```jsx
// Example of a React / Reach Router Link component.
// Location prop takes the browser's current location.

function Navlink({ text = "Navlink", to = "/posts", location = "/" }) {
  return (
    <Link
      className="nav-link"
      data-is-active={location.pathname === to}
      to={to}
    >
      {text}
    </Link>
  )
}
```
